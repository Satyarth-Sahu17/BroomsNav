# 🧹 START HERE - BroomNav Quick Launch Guide

## 🚀 3 Steps to Launch Your App

### STEP 1: Open Terminal
- **Windows**: Press `Win + R`, type `cmd`, press Enter
- **Mac**: Press `Cmd + Space`, type `terminal`, press Enter
- Navigate to this folder: `cd [path-to-BroomNav]`

### STEP 2: Start Server
```bash
npm start
```

Wait for this message:
```
🧹 BroomNav server running on http://localhost:3000
```

### STEP 3: Open Browser
Open any browser and go to:
```
http://localhost:3000/login.html
```

## 📱 Install on Your Phone

### Android:
1. Open `http://[your-computer-ip]:3000` on phone
2. Chrome menu (⋮) → "Add to Home Screen"
3. Done\! Use like a native app

### iOS:
1. Open in Safari
2. Share button → "Add to Home Screen"
3. Done\! Use like a native app

## ❓ Troubleshooting

**Port 3000 already in use?**
→ Kill the process or change PORT in .env

**Map not loading?**
→ Check internet connection

**No suggestions appearing?**
→ Select city first\!

**Can't find location?**
→ Use area names from the city (check indian-cities.js)
