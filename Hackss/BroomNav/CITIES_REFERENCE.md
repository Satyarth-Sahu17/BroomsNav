## 💡 Usage Tips

1. **Type area name exactly**: "Bandra" not "bandra west"
2. **Use autocomplete**: Start typing for suggestions
3. **Select city first**: Required before searching
4. **Case-insensitive**: "Andheri" or "andheri" both work
5. **No need for full address**: Just area name is enough

## 🗺️ Adding More Areas

To add areas to a city:

1. Edit `indian-cities.js`
2. Add to areas array:
```javascript
{ name: "New Area", lat: 12.3456, lng: 78.9012 }
```
3. Get coordinates from Google Maps

## 🎓 For Developers

### City Object Structure:
```javascript
cityname: {
    name: "Display Name",
    center: [latitude, longitude],
    zoom: 12,
    areas: [
        { name: "Area Name", lat: x.xxxx, lng: y.yyyy },
        // ... more areas
    ]
}
```

### Safety Zone Structure:
```javascript
cityname: {
    highRiskZones: [
        { lat, lng, radius, reason }
    ],
    moderateRiskZones: [...],
    safeZones: [...]
}
```
